## 📌 Project Overview
Garbage accumulation in public areas such as streets, parks, and bus stops creates environmental pollution and health hazards. Manual monitoring of waste is inefficient for large urban areas.

This project presents an **AI-based Garbage Detection System** using the **YOLOv5 deep learning model** integrated with a **Node.js web application** to automatically detect garbage from images and support smart city initiatives.

---

## 🎯 Objectives
- Detect garbage using deep learning
- Reduce manual monitoring efforts
- Improve cleanliness in public areas
- Provide an easy-to-use system

## 🚀 How to Run the Project

### 1️⃣ Clone Repository
```

git clone [https://github.com/YogiReddy7396/Yolo-project.git](https://github.com/YogiReddy7396/Yolo-project.git)
cd Yolo-project

```

### 2️⃣ Install Dependencies
```

pip install -r requirements.txt
npm install

```

### 3️⃣ Run the Server
```

cd yolo_project/web_app/server
node server.js

```

### 4️⃣ Open in Browser
```

[http://localhost:3000](http://localhost:3000)

```
